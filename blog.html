Fundamentals of Information Security


The goal of information security is about prevent the loss of confidentiality, loss of integrity and loss of availability for information systems and data.

Confidentiality

The confidentiality is about protecting data by unauthorized users cannot access data. This can be achieved by identifying and authentication users, and implementing access control by restrict access.

Integrity

It controls prevent any unauthorized modification of data or systems. This can be achieved by hashing and audit logging, which helps to detect alteration of data and it informs that it integrity is lost.

Availability

The data and information must be available when needed. The Fault-tolerant and redundant technologies ensure availability even when system suffers failure.
